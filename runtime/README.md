# Care Commerce Platform Runtime

Docker Compose 기반 로컬 데모 런타임입니다.

```bash
docker compose up --build
```

| Service | URL |
| --- | --- |
| Backend health | http://localhost:8080/client-api/v1/health |
| Admin Web | http://localhost:4174 |
| Desktop Web | http://localhost:4176 |
| PostgreSQL | localhost:15432 |
