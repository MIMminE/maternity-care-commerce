# Requirements

## 사용자 기능

- 회원가입/로그인
- 산모 프로필 등록
- 임신 주차별 콘텐츠 조회
- 상품 목록/상세
- 장바구니/주문
- 결제 요청/결제 결과 확인
- 배송 조회
- 상품 리뷰 작성
- 상품 Q&A 작성
- 상담 신청
- 알림 확인

## 관리자 기능

- 회원 검색
- 산모 프로필 조회
- 상품 관리
- 주문/배송 관리
- 상담 처리
- 상품 Q&A 처리
- 리뷰 관리
- 쿠폰 관리
- 마케팅 동의 고객 필터링
- 개인정보 접근 로그 조회
- 운영 통계 조회

## 1차 구현 우선순위

1. 회원/인증과 관리자 권한
2. 산모 프로필과 동의 이력
3. 상품/장바구니/주문
4. 상담/문의와 관리자 처리
5. 감사 로그와 운영 통계
