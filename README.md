# Maternity Care Commerce

임산부와 산모를 위한 모성 케어 플랫폼, 커머스, 관리자 운영 시스템을 통합한 포트폴리오 프로젝트입니다.

## 실행 단위

- `backend`: Spring Boot API server
- `client-web`: 사용자용 모바일 웹/PWA
- `admin-web`: 관리자용 백오피스 웹

## 주요 목표

- 사용자 서비스와 관리자 운영 시스템의 API/권한 분리
- 일반 커머스 구조에 모성 케어 도메인 확장
- 산모 프로필, 동의 이력, 개인정보 접근 로그 설계
- AWS 수동 인프라 구성과 GitHub Actions 기반 CI/CD

## 로컬 포트

- Backend: `http://localhost:8080`
- Client Web: `http://localhost:5173`
- Admin Web: `http://localhost:5174`
